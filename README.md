# Battle City Tank Game

A Python implementation of the classic Battle City (坦克大战) game.

## Requirements

- Python 3.x
- pygame

## Installation

```bash
pip install -r requirements.txt
```

## How to Play

### Player 1 Controls
- W: Move Up
- S: Move Down
- A: Move Left
- D: Move Right
- J: Fire

### Player 2 Controls
- ↑: Move Up
- ↓: Move Down
- ←: Move Left
- →: Move Right
- Space: Fire

## Game Features

- Multiple levels with different maps
- Various enemy tank types (basic, fast, power, heavy)
- Power-ups (star, grenade, helmet, shovel, tank, clock)
- Two-player support

## Run the Game

```bash
python main.py
```
